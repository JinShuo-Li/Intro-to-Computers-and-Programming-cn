# Cheating Sheet of Intro. to Computers and Programming

## English Version

### Overview
This repository contains a comprehensive final review cheating sheet for the course "Introduction to Computation with Python Programming". The document is written in LaTeX and covers essential topics from Linux commands to advanced Python concepts, designed to help students prepare for exams efficiently.

### Key Topics Covered
- Linux & Python Common Commands
- Basic Python Types and Operations
- Variables and Assignments
- Conditional Branching
- Loops
- Functions
- Built-in Functions
- Lists, Tuples, Dictionaries, Sets
- Strings
- Object-Oriented Programming (Classes)
- Modules
- Exceptions and Assertions
- Random Module
- File Operations
- Iterators and Unpacking
- Structural Pattern Matching
- Generators
- Advanced Function Features

### How to Use
1. **Compile the LaTeX File**: Use a LaTeX compiler like pdflatex or XeLaTeX (recommended for Chinese support) to generate the PDF.
   ```
   xelatex cheating_sheet.tex
   ```
2. **Dependencies**: Ensure you have packages like `ctex`, `geometry`, `amsmath`, etc., installed via TeX Live or MiKTeX.
3. **View PDF**: Open the generated `cheating_sheet.pdf` for the formatted review notes.
